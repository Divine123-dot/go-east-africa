Go East Africa - Travel Agency Project
Project Overview
This is a multi-page web application designed to showcase travel destinations in East Africa, focusing on cultural heritage, sightseeing, and local experiences. The project was built using semantic HTML5 and custom CSS3 to provide a professional user experience.

Key Features
Multi-Page Navigation: A global navigation bar linking five distinct sections: Home, About, Sightseeing, Things to Do, and Information.

Responsive Layouts: Utilizes advanced CSS techniques like floats (diani-float, mungwi-float) for magazine-style text wrapping.

Interactive Forms: A "Contact Us" page featuring a form with email validation and required input fields.

Asset Management: Organized directory structure for over 15 high-quality images of East African landmarks.

Technologies Used
HTML5: For semantic structure (nav, main, section, footer).

CSS3: For custom styling, layout management, and visual consistency.

Git/GitHub: For version control and deployment via the Linux terminal.
